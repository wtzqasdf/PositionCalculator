# 倉位計算機(PositionCalculator)
此計算機使用於加密貨幣，它可以幫助你計算每個倉位/所有倉位的盈虧

## 程式功能
1. 即時價格來源為Binance
2. 支援一個交易對的計算
3. 自動儲存輸入的倉位資料，下次開啟瀏覽器它還會在
4. 詳細的盈虧計算表示
